# FCC Process Analysis

Statistical process monitoring and time-series analysis on one year of synthetic FCC sensor data — applying NumPy, Pandas, and visualization to the kind of questions a process operator asks every shift.

---

## 📦 Dataset

This project uses the **synthetic FCC dataset** generated by [synthetic-fcc-dataset](https://github.com/JOUW-USERNAME/synthetic-fcc-dataset) — 105,120 rows of process data at 5-minute intervals over one full year.

The dataset simulates realistic FCC operations including catalyst activity degradation, seasonal air flow variation, process disturbances, and PID control behavior. See that repository for full column definitions and generation methodology.

**Dataset:** `fcc_dataset_large.csv` (~50MB, not tracked in this repo)

To reproduce this analysis, generate the dataset first:
```bash
# In the synthetic-fcc-dataset repo
python generate_dataset.py
# Then copy fcc_dataset_large.csv into the data/ folder of this repo
```

---

## 🎯 Analysis Goals

This project asks the same questions a shift supervisor would ask — but answered with data instead of gut feel:

- Which process variables are most variable, and is that variability expected or concerning?
- Where are the statistical outliers, and do they cluster around known events (catalyst replacement, disturbances)?
- What does "normal operation" look like as a statistical envelope — and how often does the process run outside it?
- How do correlated variables like `Reactor_Temperature` and `Conversion_Rate` behave across time?
